## 프로젝트 소개

**Traflix**는 출발지와 도착지를 선택하면, 이동 경로간에 사용자의 취향에 맞는 기차역 주변 여행지 [관광지, 문화시설, 축제/공연, 카페, 음식점]을 추천해주고, 최적의 동선[연계 열차]을 제안해주는 웹 서비스입니다.
